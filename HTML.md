# HTML이란?

_: 웹페이지의 구조와 내용을 정의하는 표준 마크업 언어이다._

> **HTML**은 웹 개발의 **필수 기초 언어**이며, 단순히 텍스트를 나열하는 것이 아니라 "_문맥을 부여하는 역할._"

**주요 기능**

- 웹 콘텐츠의 의미와 구조 정의.

**활용 예시**

- 제목 설정: `<h1>제목</h1>`
- 문단 작성: `<p>내용</p>`
- 이미지 삽입: `<img src="image.jpg">`
- 링크 연결: `<a href="url">링크</a>`  
  ::_텍스트, 이미지, 링크, 버튼 등을 어디에 배치할지 표시._

HTML과 관련된 대표적인 언어는  
**CSS, JavaScript** 등이 있다.

> HTML이 담당하는 일을 더 자세하게 알아보자.

#### 웹 사이트에서 HTML의 담당은?

- **문서 구조 만들기.**  
  :ex) 제목, 문단, 목록
- **링크 만들기.**
- **이미지 넣기.**
- **폼 만들기.**  
  :ex) 로그인, 검색창, 입력창.
- **미디어 표시.**

_...같은 역할을 맡고있다._

### HTML의 기본 개념

1. **태그 (Tag)**

- HTML은 태그로 구성됨.
  :ex) `<p>` 시작,
  `</p>` 끝.

1. **속성 (Attribute)**

- 태그에 추가 정보를 줌.

1. **중첩 (Nesting)**

- 태그안에 태그 가능.

> 기본 태그를 알기 전, 기본 구조를 알아보자.

**HTML의 기본구조**

`<!DOCTYPE html>`  
`<html>`  
`<head>`  
 ` <title>페이지 제목</title>`  
`</head>`  
`<body>`

`</body>`  
`</html>`  
_기본구조는 !를 통해 빠르게 작성 가능하다._

## HTML의 기본 태그

- 제목 태그  
  `<h1>제목</h1>`  
  `<h2>제목</h2>`  
  `<h3>제목</h3>`  
  _`<h1>`가 제일 큼._

- 문단  
  `<p>문장입니다.</p>`

- 줄바꿈  
  `<br>`

- 링크  
  `<a href="(링크)">링크</a>`

- 이미지  
  `<img src="사진">`

- 목록
- 순서 없는 목록  
  ` <ul>`  
  `<li>가</li>`  
  `<li>나</li>`  
  `</ul>`
- 순서 있는 목록  
   `<ol>`  
  `<li>다</li>`  
  `<li>라</li>`  
  `</ol>`

- 강조
  `<b>굵다</b>`  
  `<i>기울인다</i>`
